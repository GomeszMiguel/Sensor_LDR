Descrição detalhada

Implementação de leitura de luminosidade utilizando um sensor LDR em um sistema Linux embarcado.

Autor
Miguel Gomes
Data

Este programa lê valores do ADC exportados no sysfs e converte a resistência do LDR em uma estimativa percentual de luminosidade (0% = escuro, 100% = claro).
