Descrição detalhada

Implementação de leitura de luminosidade utilizando um sensor LDR em um sistema Linux embarcado.

Autor
Miguel Gomes
Data
22 de Outubro de 2025
  • Este programa lê valores do ADC exportados no sysfs e converte a resistência do LDR em uma estimativa percentual de luminosidade (0% = escuro, 100% = claro). O valor lido é então enviado via protocolo UDP (datagrama) para o servidor rodando no endereço SERVER_IP (Host Windows/WSL) na porta 8080.

Função principal.

Configura o socket UDP para enviar dados para o servidor 192.168.42.10:8080. Cria um objeto SensorLDR, lê continuamente a luminosidade, converte o valor inteiro para string e envia o datagrama para o servidor a cada segundo.

Retorna
0 em caso de execução normal.
